## Building
Requirements for Windows
- [git](https://git-scm.com/)
- [CMake](https://cmake.org/) 3.17.5 or newer
- [Visual Studio 2019](https://visualstudio.microsoft.com/vs/)

1. `git clone --recurse-submodules https://github.com/Link1J/TNAE.git`
2. `cd TNAE`
3. `mkdir build`
4. `cd build`
5. `cmake ..`
6. `cmake --build .`
